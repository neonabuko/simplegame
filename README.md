<h2>Installing</h2>

```shell
git clone https://github.com/neonabuko/simplegame

cd simplegame

mkdir build && cd build

cmake ..

cmake --build .
```

<h2>Running</h2>

```shell
cd simplegame/build/

./simplegame
```
